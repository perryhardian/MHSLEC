# E2E Checklist

Gunakan checklist ini untuk validasi manual di Android emulator setelah backend dan APK terbaru berjalan.

## Environment

- Date:
- Tester:
- Backend URL:
- APK build:
- Emulator:

## Auth

- [ ] Register user baru berhasil.
- [ ] User masuk ke Dashboard setelah register.
- [ ] Logout berhasil.
- [ ] Login ulang berhasil.
- [ ] Session tetap valid saat app dibuka ulang.

## Transactions

- [ ] Tambah expense berhasil.
- [ ] Tambah income berhasil.
- [ ] Edit transaksi berhasil.
- [ ] Hapus transaksi berhasil.
- [ ] Dashboard berubah setelah transaksi berubah.

## Budgets

- [ ] Tambah budget berhasil.
- [ ] Edit budget berhasil.
- [ ] Hapus budget berhasil.
- [ ] Progress budget berubah setelah transaksi expense ditambah.

## Savings Goals

- [ ] Tambah savings goal berhasil.
- [ ] Setor tabungan berhasil.
- [ ] Edit savings goal berhasil.
- [ ] Hapus savings goal berhasil.
- [ ] Status completed/cancelled tampil benar.

## Reports

- [ ] Laporan bulanan terbuka.
- [ ] Navigasi bulan sebelumnya/berikutnya berhasil.
- [ ] Summary laporan sesuai transaksi.
- [ ] Category breakdown tampil.

## Notifications

- [ ] Generate notifications berhasil.
- [ ] Badge unread tampil.
- [ ] Mark as read berhasil.
- [ ] Mark all as read berhasil.
- [ ] Delete notification berhasil.

## Profile And Categories

- [ ] Profile terbuka.
- [ ] Edit profile berhasil.
- [ ] Tambah kategori custom berhasil.
- [ ] Kategori custom muncul di form transaksi.
- [ ] Hapus kategori custom yang belum dipakai berhasil.

## Result

- [ ] Passed
- [ ] Passed with notes
- [ ] Failed
